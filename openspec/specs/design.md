# Technical Specification - UI Design System (Design Spec)

## 1. Color Palette (theme/colors.ts)
*   **Primary Green:** `#08a855` - Used for primary call-to-action buttons, active statuses, and positive indicators.
*   **Secondary Amber:** `#785900` - Used for gamification elements (points, level-up details), achievements, and warning icons.
*   **Background Main:** `#f1fcf1` - Eco-themed light green background to create a calming, natural environment.
*   **Surface Background:** `#ffffff` - Neutral white background for content cards and scroll views.
*   **Text Main:** `#141e17` - High contrast dark green-black for optimal readability.
*   **Text Muted:** `#58605a` - Soft grey-green for metadata, timestamps, and subtitles.

## 2. Typography System (theme/typography.ts)
*   **Font Family:** Uses system fonts (San Francisco on iOS, Roboto on Android) for maximum native performance.
*   **Hierarchy Presets:**
    *   `h1`: size 32px, Bold.
    *   `h2`: size 24px, Bold.
    *   `h3`: size 18px, Semi-Bold.
    *   `body`: size 14px, Medium.
    *   `caption`: size 12px, Regular.
