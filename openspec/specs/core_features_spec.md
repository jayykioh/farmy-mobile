# Technical Specification - Core System Features (Core Features Spec)

## 1. Feature Map
*   **Authentication:** Handles secure onboarding, register, JWT session validation, and seamless Google OAuth redirects.
*   **Crop Diary:** Interactive tracking for farm cycles (creating cycles, displaying timeline logs, completing harvests).
*   **AI Disease Scan:** Real-time leaf disease classification via device camera or gallery picking.
*   **AI Chat Assistant:** Server-Sent Events (SSE) chat client for agricultural support.
*   **Pet Gamification:** Interactive pet display, XP progression system, items shop, and layout accessories equipping.
*   **Reminders:** Scheduled farming chores list with check-off updates.
